什么是 Chrome DevTools MCP Server

MCP（MitM Chrome Protocol / Managed Chrome Proxy）Server 是一种用于管理与代理 Chrome DevTools Protocol (CDP) 会话的服务层。通过 MCP，多个客户端可以更方便地连接、复用或记录 DevTools 会话，实现远程调试、自动化测试与可视化追踪。

典型功能

  • 代理与复用 DevTools 会话
  • 记录/回放 DevTools 指令
  • 权限/多租户管理（团队协作）
  • 用于 CI 的自动化调试入口

Workshop 中的实践要点

  1. 在本地启动 MCP Server，连接 Chrome 实例
  2. 使用 DevTools 客户端复用会话进行元素检查与网络抓包
  3. 实现简单的脚本化操作（截屏、性能快照、控制台日志收集）
